Project 1: Web App – Movie Manager

A simple and easy-to-use web app that lets users keep track of the movies they've watched or want to watch, all in one place.

Technologies Used: HTML, CSS, JavaScript, React, Material UI

Description: This project is a web application that allows users to search, save, and organize their favorite movies. It features a clean and simple interface, live movie data from an external API, and a responsive design. The app is built with React and Node.js, making the experience fast and easy across all devices.
